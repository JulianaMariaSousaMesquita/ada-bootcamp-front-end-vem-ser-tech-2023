Tipos de Dados em JavaScript

JavaScript é uma linguagem de programação dinamicamente tipada, o que significa que as variáveis podem conter diferentes tipos de dados em momentos diferentes durante a execução do programa. Existem vários tipos de dados em JavaScript, e eles podem ser divididos em categorias principais, incluindo números, strings, booleanos, objetos, e valores especiais. Vamos explorar cada um deles.

Operadores Lógicos

'&&' (E lógico): Este operador retorna true se ambos os operandos forem verdadeiros. Exemplo: 

            let a = true;
            let b = false;
            let resultado = a && b; // resultado será igual a false

'||' (OU lógico): Este operador retorna true se pelo menos um dos operandos for verdadeiro. Exemplo: 

            let a = true; 
            let b = false; 
            let resultado = a || b; // resultado será igual a true

'!' (Negação lógica): Este operador inverte o valor de verdade de um operando. Exemplo: 

            let a = true; 
            let resultado = !a; // resultado será igual a false

Operadores Aritméticos

'+' (Adição): Este operador é usado para somar números ou concatenar strings. Exemplo: 

            let a = 5; 
            let b = 10; 
            let resultado = a + b; // resultado será igual a 15

Operadores Aritméticos

'-' (Subtração): Este operador é usado para subtrair números. Exemplo: 

            let a = 10; 
            let b = 5; 
            let resultado = a - b; // resultado será igual a 5

Operadores Aritméticos

'/' (Divisão): Este operador é usado para dividir números. Exemplo:

let a = 10; let b = 2; let resultado = a / b; // resultado será igual a 5

Operadores Aritméticos

'*' (Multiplicação): Este operador é usado para multiplicar números. Exemplo:

let a = 5; let b = 3; let resultado = a * b; // resultado será igual a 15

Operadores Aritméticos

'**' (Potência): Este operador é usado para calcular a potência de um número. Exemplo:

let a = 2; let b = 3; let resultado = a ** b; // resultado será igual a 8 (2 elevado à 3ª potência)

Operadores Aritméticos

'%' (Módulo): Este operador é usado para obter o resto da divisão de um número por outro. Exemplo:

let a = 10; let b = 3; let resultado = a % b; // resultado será igual a 1 (resto da divisão de 10 por 3)

Operadores Aritméticos

'++' (Incremento) e '--' (Decremento): São operadores usados para aumentar ou diminuir o valor de uma variável em 1, respectivamente.

let contador = 5; contador++; // Incrementa o contador em 1 (agora contador é 6) contador--; // Decrementa o contador em 1 (agora contador é 5 novamente)
