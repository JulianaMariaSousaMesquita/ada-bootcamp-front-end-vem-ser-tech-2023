Tipos de Dados em JavaScript

JavaScript é uma linguagem de programação dinamicamente tipada, o que significa que as variáveis podem conter diferentes tipos de dados em momentos diferentes durante a execução do programa. Existem vários tipos de dados em JavaScript, e eles podem ser divididos em categorias principais, incluindo números, strings, booleanos, objetos, e valores especiais. Vamos explorar cada um deles.

Formatação de Números

JavaScript permite formatar números para melhor legibilidade. Você pode usar sublinhados para separar grupos de dígitos em números inteiros e notação exponencial para lidar com números muito grandes ou muito pequenos. 

    
let numeroGrande = 1_000_000; // 1 milhão
let numeroExponencial = 3.0e8; // 300 milhões (notação exponencial)

Entender os tipos de dados em JavaScript é fundamental para desenvolver aplicativos eficazes. Cada tipo de dado tem seu papel específico e características próprias, permitindo que você modele e manipule informações de maneira eficiente em seus programas.

Array em JavaScript

Indexação numérica

Os elementos de um array são acessados usando índices numéricos:

          
      let meuArray = [1, 2, 3];
      console.log(meuArray[0]); // 1

Métodos específicos do array

Os arrays possuem métodos embutidos para manipulação e iteração:

          
      let meuArray = [1, 2, 3];
      meuArray.push(4);
      console.log(meuArray); // [1, 2, 3, 4]

Propriedade length

Os arrays têm uma propriedade length para obter o número de elementos: 

          
      let meuArray = [1, 2, 3];
      console.log(meuArray.length); // 3

2. BigInt

O tipo BigInt foi introduzido em JavaScript para lidar com números inteiros grandes que não podem ser representados com precisão usando números regulares. 

    
let numeroGrande = 1234567890123456789012345678901234567890n;

3. Strings

As strings representam sequências de caracteres e podem ser criadas usando aspas simples ou duplas. 

    
let stringComAspasSimples = 'Olá, mundo!';
let stringComAspasDuplas = "JavaScript é incrível!";

4. Booleanos

Os booleanos representam valores verdadeiros ou falsos e são frequentemente usados em avaliações condicionais. 

    
let verdadeiro = true;
let falso = false;

5. Null e Undefined

null é usado para representar a ausência intencional de valor, enquanto undefined indica que uma variável foi declarada, mas ainda não foi inicializada. 

          
      let valorNulo = null;
      let valorIndefinido;

6. NaN e Infinity

NaN (Not-a-Number) é usado para representar resultados inválidos em operações matemáticas, enquanto Infinity representa valores infinitos positivos e -Infinity representa valores infinitos negativos. 

          
      let resultadoInvalido = 10 / "abc"; // NaN
      let infinitoPositivo = 1 / 0;      // Infinity
      let infinitoNegativo = -1 / 0;     // -Infinity

7. Symbol

Symbol é um tipo de dado único e imutável que é usado para criar identificadores exclusivos. Cada Symbol é único, mesmo que seus valores sejam idênticos. 

          
      const simbolo1 = Symbol('descricao');
      const simbolo2 = Symbol('descricao');
      console.log(simbolo1 === simbolo2); // false, porque são símbolos diferentes

1. Números

Os números em JavaScript podem ser inteiros ou de ponto flutuante. Você pode usar operadores matemáticos padrão para realizar cálculos. 

    
let numeroInteiro = 42;
let numeroPontoFlutuante = 3.14;
let numeroGrande = 123_456_789; // Notação com sublinhado para melhor legibilidade
